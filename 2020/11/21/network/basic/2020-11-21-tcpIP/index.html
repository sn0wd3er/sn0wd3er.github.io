TCP/IP

현재 쓰고 있는 네트워크 기본 구조이다. 오픈 스펙이지만 보안부분에서는 안좋다.

TCP Header

Header는 최소 20byte이다.

UDP Header

header는 8byte이다.

TCP와 UDP의 비교

TCP UDP

IP
