무선 통신

Author: Sn0wd3er Date: November 16, 2020  14:39:16 Category: Network

무선 통신

무선 통신은 전자기파를 이용한 통신 방법과 이용한 초음파 영역을 이용한 통신 방법이다.

주파수

주파수는 1초에 몇 번 떨리는지를 나타낸 것

  • Hz: 주파수 단위

주파수는 파장과 반비례하며 낮은 주파수는 파장이 커지므로 대역폭이 증가한다. 반대로 높은 주파수는 파장이 낮아서 대역폭이 짧아지지만 데이터를 빠르게 전송시킬 수 있다.

WAP

무선 인터넷 프로토콜로 무선 장치를 이용하여 인터넷 서비스를 이용하도록 지원하는 표준 기술이다.

구성

WAP 게이트웨이

유 - 무선을 연결시켜주는 인터페이스 역할을 하며 HTML 콘텐츠를 WML로 변환하는 등 인터페이스 역할을 한다.

WAE

다양한 무선 환경에 대한 응용 계층 상호운용성 제공, 서로 다른 단말기 끼리 통신할 수 있게 도와준다. Application

WSL

연결지향/비연결지향 세션 관리 기능을 제공한다. Seesion

WTP

트랜잭션 서비스 제공, 무선과 유선 네트워크에서 TCP 프로토콜 간의 연결을 해준다. Transaction

WTLS

전송 계층의 보안 서비스 제공, TLS기반으로한 보안 프로토콜. Security

  • Record Protocol

  • Handshake Protocol

    • 세션 동안 공유 비밀 정보, 세션키의 생성 및 공유 지원

  • Change cipher specs Protocol

    • 보안 연결의 암호화 스펙과 시작을 통보하기 위해 사용

  • Alert Protocol

    • 세션 중에 발생하는 오류를 알림

      WDP

무선 데이터 송 - 수신 기능 제공, Transport

WPKI

무선 환경에서 공개키 기반 인증, 키 분배를 제공하는 구조, 모바일 뱅킹등 공인인증서 에서 많이 쓰임

구성

CP 서버

무선 콘텐츠를 제공, 인증서 관리 및 검증 요청을 대행하는 등록기관 역할을 수행

OCSP 서버

인증서 상태 정보를 실시간으로 제공

CA

WPKI 무선 인증 기관 , 인증서 생성, 발급, 갱신, 폐지 등의 서비스를 제공

MS

핸드폰이나 PDA같은 무선 단말기를 뜻함
